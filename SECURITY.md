# Security Policy

## Supported Versions

We actively monitor and support the latest release of the toolbox. Security updates and bug fixes will be applied directly to the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| Current | :white_check_mark: |
| < Prior | :x:                |

## Reporting a Vulnerability

We are committed to preserving the integrity and safety of this toolbox. If you discover a security vulnerability, please do **not** open a public issue or pull request. 

Instead, please utilize GitHub's native private disclosure feature:

1. Navigate to the [Security](https://github.com/sfu-cs-vision-lab/individual-cmfs-matlab/security) tab at the top of the repository page.
2. Click on **Advisories** in the left sidebar.
3. Click the **Report a vulnerability** button.

This opens a private workspace where you can safely share details, reproduction steps, and impact vectors directly with the repository maintainers without exposing the vulnerability prematurely to the public. 

We track incoming alerts actively and appreciate your help in keeping the color and vision research community secure!
